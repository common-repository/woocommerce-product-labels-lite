With Woocommerce Product Labels Lite you can insert product images with labels on them. Any type of content can be added.

== Description ==
Forget about old style text descriptions of your products! With Woocommerce Product Labels plugin you can add description of product elements directly to images. Make your descriptions stylish and increase your sales!
Plugin is fully responcive and optimized for moblie devices, so you can sell items for via any mobile devices.

You can easily customize position of labels, pick colors, icons etc to make all labels look 100% cool!

Plugin have ready to go .po files in packeage so they can be easily translated to any language.

== Installation ==

1. Unzip the plugin and upload it to your site's wp-content/plugins folder.
2. Activate Woocommerce Product Labels Lite through the "plugins" area in your WordPress dashboard.
3. In Woocommerce product editor you will find new meta box - Edit Labels. There you can add/remove/edit all labels.
